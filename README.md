## Code 작성 규칙
### Commit Message
- feat : 새로운 기능 추가/수정/삭제
- enhance : 기존 코드에 기능을 추가하거나 기능을 강화할 때
- refactor : 코드 리팩토링,버그 수정
- test : 테스트 코드/기능 추가
- edit : 파일을 수정한 경우(파일위치변경, 파일이름 변경, 삭제)
- chore : 기존 코드 변경(불필요한 코드 삭제 등)

### Naming
- Class : Pascal 
- Variable : Snake 
- Function : Snake 
- Constant : Pascal + Snake
